About 8Builders4Tools (8b4t)

8b4t is a Minecraft Java Edition anarchy server that first opened on January 6th, 2024. It has no map resets and no rules, allowing players to do whatever they want.

The server supports cracked players with a /register and /login plugin system. The owner of 8b4t is OG_Minecrafter.

The server stays close to vanilla with minimal modifications. Hack clients and cheats are allowed, but a light anticheat and exploit fix plugin prevent game-breaking exploits.

Server IP: 8b4t.com

Hosted on 1.21.4, but older clients 1.8+ can still join (with some limitations).
